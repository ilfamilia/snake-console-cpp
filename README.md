# Snake Game (C++ Console)

A simple Snake game implemented in C++ using console rendering.

## Features
- Continuous snake movement
- Direction control using WASD
- Random food spawning
- Food respawns after being eaten
- Score tracking displayed on screen
- Collision detection with walls
- Clean console rendering with separated UI

## Controls
- W: Move up
- S: Move down
- A: Move left
- D: Move right

## Build
This project uses CMake.

## Notes
- The game runs in the Windows console.
- Rendering and game logic are separated into distinct components.
